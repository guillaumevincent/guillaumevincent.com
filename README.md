# guillaumevincent.com


## Requirements
* [jekyll](http://jekyllrb.com/)
* [nodejs](http://nodejs.org/)
* [gulp](http://gulpjs.com/)

## Installation

run

    npm install

build static files

    gulp build

## Build site and run jekyll

run 

    jekyll serve

and visit http://0.0.0.0:4000/
